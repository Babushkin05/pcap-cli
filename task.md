# Задание HW3 - Реализация DNS функций для pcap-cli

## Цель
Расширить существующий проект pcap-cli (HW2) новыми DNS-функциями в соответствии с требованиями HW3.

## Требования из HW3
1. Захват всех пакетов DNS и вывод их на консоль
2. Поиск IP-адреса почтового сервиса (MX запись) по доменному имени
3. Обращение к корневому DNS-серверу для поиска IP-адресов доменов
4. Сравнение ответов корневого сервера и DNS-сервера провайдера
5. Использование PCAP (низкоуровневый API)

## Реализованные функции
- [x] Команда `dns-sniff` - захват и вывод DNS-пакетов
- [x] Команда `dns-lookup` - поиск DNS-записей (пока только MX)
- [x] Команда `dns-root` - запрос к корневым DNS-серверам
- [x] Полный парсинг DNS-пакетов (все секции: вопросы, ответы, авторитетные, дополнительные)
- [x] Отправка DNS-запросов через PCAP
- [x] Получение и обработка DNS-ответов

## Архитектура
- В `internal/dns/` - логика работы с DNS-протоколом
- В `internal/commands/` - реализация команд CLI
- В `cmd/pcap/` - точка входа и корневая команда

## Статус
Проект собирается и готов к тестированию. Все основные функции DNS-протокола реализованы в соответствии с требованиями HW3.
